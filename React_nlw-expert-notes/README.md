# Trilha ReactJS - Aplicativo Notes
### React + TypeScript + Vite + TailwindCSS

Aplicativo de notas front-end, podendo ser **digitado** ou utilizada a função **"Voz para texto"** e possibilidade de busca de conteudo nas notas.
